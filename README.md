# USDC Subgraph (Ethereum Mainnet)

This subgraph indexes all **Transfer events** from the USDC token contract.

## Commands

```bash
# Install deps
npm install

# Generate types
npm run codegen

# Build
npm run build

# Deploy (hosted service)
npm run deploy
```
